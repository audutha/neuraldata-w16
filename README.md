neuraldata-w16
==============

MATLAB code for the analysis of neural data, used in graduate course
"Analysis of Neural Data" at Dartmouth.

please see
[here](https://github.com/mvdm/vandermeerlab/blob/master/LICENSE.md)
for licensing information.
